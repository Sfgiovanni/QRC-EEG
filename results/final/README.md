# Final machine-readable inputs

These files are derived from canonical Gate 0–3 sources by
`scripts/build_repository_release.py`. They are intended to supply traceable numbers and CSV
tables to later human manuscript work. They are not a manuscript and contain no LaTeX.

Use `key_results.csv`, `claims_registry.csv`, the six `table_*.csv` files and the repository
release report. Rebuild and verify with `bash scripts/run_repository_release.sh`.
