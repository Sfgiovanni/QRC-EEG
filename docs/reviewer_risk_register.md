# Reviewer risk register

| Risk | Severity | Available evidence | Honest response | Avoid |
|---|---|---|---|---|
| No quantum advantage | High | Strong classical controls; short-horizon classical lead | Frame a simulated mechanism and horizon interaction | “quantum advantage” |
| Hybrid density-matrix mechanism | High | Resource audit and recurrence code | Call it hybrid/simulated; state hardware requirements | “immediately NISQ-ready” |
| Single EEG database | High | Bonn provenance | Limit inference to benchmark segments | clinical/general EEG claims |
| Segment, not patient | High | Frozen splits and protocol | State no subject mapping or subject-disjoint split | `n_patients` or patient generalization |
| Temporal autocorrelation | High | Causal preprocessing and segment-blocked validation | Explain what leakage controls do and do not solve | iid-row inference |
| AR/ESN comparisons | Medium | Frozen gate curves and interactions | Report classical short-horizon lead and exact scope | absolute superiority |
| Kernel forms tied | High | Curves/useful horizons | Claim distributed class only where supported | exponential universality |
| Gate 2 moderate within scenario | High | centered, pairwise and LOSO addendum | Separate between-process scale from within-process ordering | universal prediction law |
| High shot cost/tail | High | resource and tail CSVs | Report mixed sensitivity and conservative preparations | global robustness |
| h=64 no absolute skill | High | NRMSE curves | Use as frozen interaction endpoint only | long-horizon forecasting headline |
| State-history implementability | High | no-cloning/backaction audit | Require explicit reproduction/memory protocol | free state buffer |

This register is a writing guardrail, not a rebuttal template. Scientific negatives remain valid
results and must not be reframed as execution failures.
